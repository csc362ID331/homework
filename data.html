NOIR - 4 Data Types to Know

Nominal

Nominal variables are those which represent qualitative variables. These varaibles are represented purely by a name and do not have any values associated with them

Ordinal

Ordinal variables are those which have an order but one in which the intervals between values is not clearly defined.

Interval

Interval variables are those whiich have an order as well as a measurable difference between values. Interval data do not have an absolute zero.

Ratio

Ratio data are similar to interval data but they do have an absolute zero. This quality makes them the most appealing type of data to work with.
